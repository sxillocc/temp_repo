Atomic Package
AtomicInteger, AtomicBoolean-

int, bool and other primitive data type not works properly when being shared by multiple threads, sometimes if multiple threads do add operation together without any difference in moment of execution, then there will be single update instead of twice update.
Atomic Package data type help us to update particular variable under lock, in a synchronized manner
